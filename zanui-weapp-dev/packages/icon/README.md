## Icon 图标

### 使用指南
在 index.json 中引入组件
```json
{
  "usingComponents": {
    "zan-icon": "path/to/zanui-weapp/dist/icon/index"
  }
}
```

### 代码演示
可以在任意位置上使用 zan-icon 标签。通过 type 可以控制 icon 显示的图标
```html
<zan-icon type="success"></zan-icon>
```
